# Automat do kawy z funkcją profilowania

Projekt uczelniany dotyczący inteligentnego automatu do kawy z funkcją personalizacji obsługi użytkownika.

## Opis projektu

Celem projektu jest opracowanie koncepcji systemu automatu do kawy z dodatkowymi funkcjami wspierającymi wygodę użytkowania oraz personalizację obsługi.

Projekt obejmuje analizę wymagań, przygotowanie dokumentacji oraz tworzenie modeli i diagramów w Enterprise Architect.

## Dokumentacja

W repozytorium znajdują się materiały projektowe, w tym dokumentacja, wymagania systemowe oraz diagramy związane z projektem.

Dokumentacja opisuje między innymi funkcje użytkownika, wymagania funkcjonalne i pozafunkcjonalne oraz środowisko docelowe systemu.
